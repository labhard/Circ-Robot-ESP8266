# Circ-Robot-ESP8266
Printed circit board to host a NodeMCU in a R/C mobile robot. The intended application is in MINI robot soccer of robocore.net events.
Ciruito impresso para suportar um NodeMCU para robô móvel de R/C. A aplicação de interesse é para equipar robôs em competições futebol categoria MINI dos eventos da Robocore.

Este projeto foi criado utilizando Kicad 8.
A fonte de alimentação recomendada: bateria de lítio 2S. Exemplo: duas células 18650 em série. Não há carregador embarcado.
O controle dos motores é feito com uma ponte H usando circuitos integrados como o MX1508 ou MX1616. Modelo de referência: https://www.robocore.net/driver-motor/mini-ponte-h-dupla-mx1616
